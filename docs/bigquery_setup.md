# Google BigQuery Setup

## Objective

Google BigQuery was selected as the cloud data warehouse for this project.

The objective is to store retail sales data in a scalable cloud environment suitable for SQL analysis and forecasting.

## Steps Completed

- Created Google Cloud Project
- Enabled BigQuery API
- Created BigQuery Dataset
- Configured cloud environment for data loading

## Benefits

Using BigQuery provides:

- Scalable cloud storage
- High-performance SQL queries
- Integration with dbt
- Efficient feature engineering
- Support for large datasets

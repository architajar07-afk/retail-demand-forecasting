# Retail Demand Forecasting & Inventory Optimization

## Project Overview

This project aims to predict future retail product demand using historical Walmart sales data from the M5 Forecasting Dataset.

The objective is to build an end-to-end analytics pipeline including:

- Data exploration
- Data transformation
- Time-series forecasting
- Inventory optimization
- Interactive Streamlit dashboard
